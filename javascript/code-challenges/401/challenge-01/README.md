# Reverse an Array

### Feature Tasks

Write a function called reverseArray which takes an array as an argument. Without utilizing any of the built-in methods available to your language, return an array with elements in reversed order.

### Expected Output

Start [1, 2, 3, 4, 5, 6]  
End [6, 5, 4, 3, 2, 1]
